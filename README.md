# Insertion Sort Projesi

Bu proje, Insertion Sort algoritmasını açıklamak ve uygulamak için yapılmıştır.

Verilen Dizi: [22, 27, 16, 2, 18, 6]

1. [22, 27, 16, 2, 18, 6]
2. [16, 22, 27, 2, 18, 6]
3. [2, 16, 22, 27, 18, 6]
4. [2, 16, 18, 22, 27, 6]
5. [2, 6, 16, 18, 22, 27]

Big-O Gösterimi:

- **Average Case**: O(n^2)
- **Worst Case**: O(n^2)
- **Best Case**: O(n)

18 Sayısının Durumu:

- **Average Case**: 18 sayısı ortada yer alır.
